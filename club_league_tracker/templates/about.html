About this site

This site is intended to be used for the IX Electron club.

It is developed and maintained by a sole developer so functionality is limited.
For feature requests, bugs, comments, contact me on Discord. You know who I am.

This website was born because BS currently does not provide an easy way to track club participation.
It does not even do so through their official API since it doesn't provide club league data.

The goal of this website is to help club leadership keep track of club member activity
throughout club league in a conventient and simple way.

This website is intended to be temporary
until BS decides to actually implement a club league API.
